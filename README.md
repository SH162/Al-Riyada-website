# شركة الريادة | استيراد السيارات وقطع الغيار

موقع ثابت احترافي لشركة الريادة لاستيراد السيارات وقطع غيار السيارات.

## هيكل المشروع

```
Aryada/
├── index.html      # الصفحة الرئيسية
├── css/
│   └── style.css   # ملف التصميم الرئيسي
├── js/
│   └── main.js     # التفاعلات البسيطة
└── README.md       # التوثيق
```

## التشغيل

1. افتح الملف `index.html` في المتصفح مباشرة
2. أو استخدم خادم محلي للمطورين:
   ```bash
   # باستخدام Python
   python -m http.server 8000
   
   # أو باستخدام Node.js (npx)
   npx serve
   ```

## المميزات

- ✅ تصميم ثابت (بدون قاعدة بيانات)
- ✅ كود منظم ونظيف
- ✅ دعم كامل للغة العربية (RTL)
- ✅ متجاوب مع جميع الأجهزة
- ✅ واجهة عصرية وجذابة

## التخصيص

يمكنك تعديل:
- **الرقم والبريد**: في قسم التواصل في `index.html`
- **الألوان**: في المتغيرات ببداية `css/style.css`
- **النصوص**: مباشرة في `index.html`
